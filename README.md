Welcome to the **Open Source Handbook**! This is a training repository designed for students who are new to GitHub and Open Source.

## Goal of the Activity
Your goal is to explore this codebase and identify issues that need fixing. These issues range from simple typos to minor design flaws.

### How to Participate
1.  **Explore**: Open `index.html` in your browser.
2.  **Identify**: Look for things that don't look right.
3.  **Document**: Write down:
    - **Repo Name**: (The name of this repository)
    - **Issue Number**: (Assign a number to each thing you find)
    - **What needs fixing**: A brief description of the bug.

## Getting Started
To view this project locally:
1. Fork this repository.
2. Clone it to your machine.
3. Open `index.html` in any browser.
