# Contributing Guide

We love contributions! Since this is a training repo, we focus on identifying and fixing "Good First Issues".

## How to Contribute
1. **Find an Issue**: Use his repo to practice finding bugs.
2. **Open a PR**: If you find an issue, feel free to submit a Pull Request with the fix!
3. **Be Kind**: Remember that everyone here is learning.

## Bug Report Template
When documenting an issue, please use the following format:
- **Issue**: Short description of the bug.
- **Location**: Which file and line number (if possible).
- **Suggested Fix**: How would you fix it?
